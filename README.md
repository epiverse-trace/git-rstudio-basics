This is a fork of the Carpentries `git-novice` lesson. This will be a derivative lesson to host the materials developed for an [Introductory training on Git and GitHub basics using Rstudio](https://epiverse-trace.github.io/learn/git-training-01/)

# git-novice

An introduction to version control for novices using Git.
Please see [https://swcarpentry.github.io/git-novice/](https://swcarpentry.github.io/git-novice/) for a rendered version of this material,
[the lesson template documentation][lesson-example]
for instructions on formatting, building, and submitting material,
or run `make` in this directory for a list of helpful commands.
